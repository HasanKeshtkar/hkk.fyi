Quick reference
  • z = Z / Z0  (normalized impedance)
  • Γ = (z − 1) / (z + 1)
  • z = (1 + Γ) / (1 − Γ)
  • SWR = (1 + |Γ|) / (1 − |Γ|)
  • Return loss = −20·log10|Γ| dB
  • y = 1/z — point diametrically opposite Γ
  • Moving ℓ toward generator: Γ·e−j4πℓ/λ (clockwise)
  • Full circle around the chart = 0.5 λ
  • Upper half: inductive (x > 0) · lower half: capacitive (x < 0)
